Hello folks. EduKart is an ongoing Open-Source project, where you can contribute to learn.

* ✔️ JWT auth.
* ✔️ Pagination support.
* ✔️ Swagger docs.
* ✔️ Written in NET.5 and EF Core 5.

### Installation

```
$ dotnet run -p Commerce.API
```
